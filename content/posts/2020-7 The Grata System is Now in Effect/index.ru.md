С момента появления этого объявления в группе концепция Системы Грата официально вступает в силу.

Система Грата является альтернативой списку неблагоприятных пользователей, поэтому постановление о чистке сообщества с текущего момента есть недействительным.

Дружба с персонами Нон Грата не может являться предлогом для исключения из группы, но может являться предлогом для создания дополнительных санкций в сторону субъектов. При этом, дружба с персонами Нон Грата может являться предлогом в отказе присоединения в группу(если пользователь не часть сообщества).

В список очков лояльности был добавлен пункт о субъективизме со стороны пользователя.

За позитивною деятельность связанную с персонами Грата вы можете получить дополнительное одобрение со стороны сообщества.
