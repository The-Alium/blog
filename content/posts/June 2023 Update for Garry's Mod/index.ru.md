Вчера вышло обновление для Garry's Mod в котором были добавлены новые функции для разработчиков, визуальные эффекты и исправления улучшающие игру.

Исправлена проблема с невозможностью захода на сервер напрямую через кнопку "Присоединиться к игре", которая была вызвана в связи с недавними обновлениями Steam. В спавнлисте были улучшены названия эффектов во вкладке текстуризации. Также в главном меню, при открытом списке аддонов, список будет сразу же обновляться при изменении подписок.

Инструмент изменения цвета подвергнут изменениям: теперь появилась возможность инвертировать цвета. Был добавлен ConVar "gmod_delete_temp_files", который будет отвечать за удаление папок materials/temp/ и sounds/temp/, при запуске и завершении игры. Также добавлено экспериментальное изменение, где полностью удалено ограничение на количество Lua файлов в режиме одиночной игры.

Исправлена неполная загрузка аддонов при присоединении к серверам где количество аддонов превышало за сотню. Исправлен вылет игры при преждевременном удалении npc_manhack. Исправлена команда toggle_duck в мультиплеере и был исправлен краш сервера при вводе команд sv_parallel_sendsnapshot и sv_parallel_packentities.

Что касается технических изменений для разработчиков, в Hammer Editor теперь появилась полная поддержка для "Static Prop Lightmap". У каждого Input/Output будет свое описание. Появились новые input/output для func_tracktrain, и шесть новых Input для всех энтити. Был добавлен keyvalue "Disable Flashlight" для всех энтити, также был добавлен prop_sphere и func_friction. Добавлены энтити filter_activator_context и filter_activator_model. Добавлены новые параметры для шейдеров "LightmappedGeneric" и "VertexLitGeneric". Были добавлены конфигурационные файлы с настройками сложности для Half-Life: Source.

Это самое основное, что можно было отметить, более подробная информация находиться на сайте Facepunch в списке изменений.
