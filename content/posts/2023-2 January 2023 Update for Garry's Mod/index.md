After seven months of inactivity in development, a small January update for Garry's Mod has been released, introducing various improvements to the game.

A new page selection button has been added in the workshop tab, allowing users to select all add-ons on a specific page. This is particularly convenient for those creating their own modification collections. The "Eye Pose" tool has also been improved based on guidance from Rubat's documentation.

The NPC weapon selection tab has been expanded with two new entries: "Shotgun" and "357. Handgun" from Half-Life: Source. Now, when an NPC kills another NPC, the kill icon displays correctly. The message "Sorry! You can't spawn that NPC!" has been removed, which previously appeared when the server did not have the requested NPC.

In terms of technical changes, the command "givecurrentammo" will now provide ammunition for alternative attacks. Additionally, when the HUD is disabled, the message "Something is creating script errors" will no longer be displayed. PrikolMen:-b has already reported a chat error when the HUD is turned off. The incorrect display of add-ons weighing 2.1 GB and more has been fixed. The server list has also been updated, which is no longer limited to 256 servers in each mode.

Numerous fixes have been made to address game crashes. In the settings, the "Legal Tab" has undergone changes, now containing links to guides on server management and modding. Lastly, the Ukrainian localization for the Trouble in Terrorist Town mode has officially been released by Erick_Maksimets, edited by Aliemann.

This covers the main highlights for casual users. More detailed information with a complete list of changes can be found on the Facepunch website.
