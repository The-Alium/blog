Спустя семь месяцев отсутствия активности в разработке выходит небольшое январское обновление на Garry's Mod добавляющее улучшения для игры.

Была добавлена кнопка выбора страницы во вкладке мастерской, позволяющая выбирать все аддоны на конкретной страницы. Удобно для тех, кто делает свои сборки модификаций. Без внимания не остался инструмент «Позер глаз», его улучшили по записям руководства Rubat.

Вкладка выбора оружия для NPC, пополнилась двумя новыми пунктами, такими как «Shotgun» и «357. Handgun» из Half-Life: Source. Теперь когда NPC убивает другого NPC, значок убийства отображается корректно. Убрано сообщение "Sorry! You can't spawn that NPC!" которое появлялось, когда у сервера нет запрошенного NPC.

Из технических изменений, теперь команда «givecurrentammo» будет выдавать патроны альтернативной атаки. Теперь, когда HUD отключен, больше не будет отображаться надпись "Something is creating script errors". PrikolMen:-b уже отрапортовал об ошибке с чатом при выключенном HUD. Было исправлено некорректное отображение аддонов с весом 2,1 GB и больше. Также был обновлен список серверов, который больше не ограничен 256 серверами в каждом режиме.

Произошли многочисленные исправления причин вылета игры. В настройках, вкладка «Legal Tab» подверглась изменениям, теперь там содержаться ссылки с руководствами по работе с серверами и моддингу. Напоследок, в режиме Trouble in Terrorist Town официально вышла украинская локализация от Erick_Maksimets за редакцией Aliemann.
